# REAL LIFE GAME CENTER (RLGC)

Life-RPG система для дисциплины и прокачки себя.

## Что внутри

- Уровни и XP
- Дейли миссии
- Ветки прокачки (Capital / Mind / Body / Influence / Style)
- Фокус-таймер (30 минут)
- Контент-трекер
- Цели и прогресс
- Система "НИЩЕТА" (анти-деградация)

## Как использовать

1. Открой сайт
2. Нажми "Новый день"
3. Закрой минимум 3 дейли

## Данные

Все данные хранятся локально в браузере (localStorage).
Для бэкапа используйте экспорт JSON
