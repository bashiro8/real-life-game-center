# real-life-game-center
Название: REAL LIFE GAME CENTER (RLGC)
Что это: оффлайн Life-RPG трекер (уровни, XP, дейли, цели, фокус, дебафы, “НИЩЕТА”).
Как пользоваться: открыть ссылку → нажать “Новый день” → закрывать дейли → качать уровни.
Данные: хранятся только в браузере (localStorage), ничего никуда не отправляется.
Бэкап: Экспорт JSON.
Перенос: Импорт JSON.
